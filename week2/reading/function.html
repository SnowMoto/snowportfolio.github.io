  1. What is a function?

    2. -It is a first-class objects that behave similar to primitive data and obects in language.
    -Can assign variables, arrays and can return by other functions.
    -There are 3 main functions that are commonly used in JavaScript:
    1. Function Declarations - declaring function().
    -ex. function hello(){console.log('Hello World!');}
    2.Function Expression - assign a anonymous function to a variable.
    -ex. const goodbye = function(){console.log('Goodbye World!');};
    3. Function Constructor - declared as an constructor for the body to be a string. BUT it is NOT recommended to use this form as there can be problems with association inside the string.
    -ex. const hi = new Function('console.log("Hi World!");');
    -Return Values - this is used to return value or it will be undefined.
    -ex. function howdy(){return 'Howdy Worl!'};
    -Parameters and Arguments - param. is a set when function is defined. Argument is the invoked function.
    -ex. parameter = function square(x){ return x*x;} - argument = square(4.5); returns = 20.25.
    -It is possible to assign a variable to the parameter without an argument.
    -ex. function hellow(name='World'){console.log('Hello ${name}!);} - invoke = hello(); return = 'Hello World!'.

  2. What are Arrow functions used for?

    3. -New syntax for declaring functions with less verbose syntax. Best used for single parameters, one line box, and return isn't required.
    -ex. const square = x => x*x;
    -ex. const hello = () => alert('Hellow World!');
    -Hoist - used to move all variables and function declarations to the top of the scope.
    -Var Hoist - moves variables to the top of the scope.
    -Callbacks - passes argument to another arument.
    -ex. function sing(song){console.log('I sing to ${song});} sing('Let it Go').
    function sing(song,callback){console.log('I sing to ${song}.'); callback();} ---given parameter then invoked inside the function.
    -If function is not provided an argument?
    -ex. if(typeof(callback) === 'function'){callback();}
