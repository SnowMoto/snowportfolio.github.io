localStorage Javascript Example

  • Week1

Information gets input by user, and web developer uses Javascript: localStorage to save information.

Can also be used for reloading what the user saved. Say they were reading a story, saved it and the next day can reload it.

There are other Javafunctions with localStorage:

-localStorage.getItem(key): Retrieves the value associated with the key.
-localStorage.removeItem(key): Removes an item from storage. The key and the value will both be removed to open space.
-localStorage.length: Returns the number of key/value pairs in the database. Usually used in a loop with the key().
-localStorage.key(i): Given an integer i, this method finds the corresponding key. This method is used in a loop to retrieve all the keys database.
Then each key is used to look up the corresponding value. -localStorage.clear(): Clears all key/value pairs from localStorage. This will wipe everything clean and nothing will be backed up.
